# GEN-PAY

A stablecoin payment processor MVP built using free-tier services:
- **Frontend:** Vercel + Next.js + v0.app UI
- **Backend:** Appwrite (Auth, DB, Functions, Teams)
- **Blockchain:** Tron (Shasta testnet) using **USDT (TRC20)**
- **Emails:** Resend
- **AI Automation:** Appwrite MCP Server

## What You'll Build

- Stablecoin **payment links**
- Tron-based **USDT settlement (TRC20)**
- **Prepaid wallet** system (merchant balance + ledger)
- **Email notifications** (receipts, alerts)
- **Webhooks** to merchant systems
- Merchant dashboard (basic ops + transaction view)
- **AI automation via MCP** (admin + ops tasks)
