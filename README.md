## üîß Installation & Setup

1. **Clone the repository**
   ```bash
   git clone https://github.com/your-username/Gen-Pay.git
   cd Gen-Pay
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Start development server**
   ```bash
   npm run dev
   ```

4. **Open in browser**
   Navigate to `http://localhost:3000`
